Div Range Slider

JQuery-less range slider with minimal JavaScript

Status: Complete
Development: Inactive
Content:

By using very little JavaScript alongside CSS and HTML, sliders with similar functionality to that of the built in range input can be created with small CSS and JavaScript file sizes. The resulting input is robust, relying on event handlers.

The slider is composed of two parts. The outer div comprises the upper track (grey), and the inner div comprises the lower track and thumb. The thumb is created with pseudo elements so that the width of the inner div can be used to represent, control, and visualize the value of the input. The outer div has the "value" attribute which will set the input's value from 0 to 100. Setting this attribute in JavaScript or after the page has loaded will not work, but it will if it is the default value before the page is loaded. The value attribute will change as the slider position is changed.
