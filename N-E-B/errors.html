Error Codes

1 - Null

When a variable is null, it has a name, but the value is 0. The error may also occur when a variable is empty (e.g. a = ""; empty, but not null or equal to 0).

2 - Length Mismatch

The Lengths of the two given variables are not equal. If this occurs when the two parents' DNA are not equal, the amount of genes that each has is not equal. In this simulation, basic values are kept the same; The cells have certain traits whose values are manipulated, and cannot be completely removed, though can be disabled if a cell has the gene set to "0". Two examples would be Flagellum Strength and Seeing distance. Cells can have a varying number of neurons in each layer of the neural network, which are set in the settings file. A version in which each cell can have a different number of neurons in each layer, and only reproduce with cells with similar neuron count is planned, but not in production. If a cell has a different number of neurons than it should, corrupt DNA, or DNA that does not follow the DNA Format being inputted by the user, this error may be thrown as a result of DNA lengths not being equal.
